## Configuration

Mesh configuration contains several sections.

* [Name](#name)
* [DataLayer](#DataLayer)
* [Endpoints](#Endpoints)
* [Proxy](#Proxy)
* [Modules](#Modules)
* [Components](#Components)

These are arranged as a set of key/value pairs on the config object:

```javascript
config = {
    name: 'mesh-name',
    dataLayer: {},
    endpoints: {},
    proxy: {},
    modules: {},
    comonents: {}
}
```
