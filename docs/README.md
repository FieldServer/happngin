## Configuration

Mesh configuration contains several sections.

* [Name](#name)
* [DataLayer](#dataLayer)
* [Endpoints](#endpoints)
* [Proxy](#proxy)
* [Modules](#modules)
* [Components](#components)

These are arranged as a set of key/value pairs on the config object:

```javascript
config = {
    name: 'mesh-name',
    dataLayer: {},
    endpoints: {},
    proxy: {},
    modules: {},
    components: {}
}
```
